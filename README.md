# Neovim config
This is my personal neovim config for the icons and devicons to work you need a nerd font perferably an monospaced so the icons dont get cut off

Package manager: Lazy.nvim

Neovim plugins:
* Treesitter
* Nvim-cmp
* Lualine
* Neovim lsp
* Nvim-autopair
* Luasnip
* Mason

Language servers:
* clangd for c/cpp
* Lua_ls for lua
